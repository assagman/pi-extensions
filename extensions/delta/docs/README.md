# Delta

Phase-gate workflow extension for Pi coding agent.

## Features

- Phase-gate development: requirements → design → plan → implement ↔ test → review → deliver
- Phase-boundary compaction for context efficiency
- Progress tracking and visualization

## Installation

```bash
npm install
npm run deploy
```

## Usage

Once deployed, use `/delta` commands in Pi.
